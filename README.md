# Student Performance Analysis Dashboard

## Overview
This project is a Power BI dashboard created using the Student Performance dataset. The dashboard performs Exploratory Data Analysis (EDA) and provides interactive insights into student performance.

## Features
- KPI Cards
- Bar Chart
- Line Chart
- Treemap
- Donut Chart
- Gauge Chart
- Interactive Slicers

## Tools Used
- Power BI
- DAX
- Power Query

## Dataset
Student Performance Dataset

## Key Insights
- Gender-wise performance analysis
- Parent education impact on scores
- Lunch distribution analysis
- Pass percentage tracking
